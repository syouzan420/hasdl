第八章 チューリングマシンとプログラムの構造

#28 チューリングマシンとコード化

ライプニッツは歯車式の計算機を設計し 製作したことでも知られる

それと同時に 人間の思考を表現する「アルファベット」の構想と そのアルファベットの組み合はせを用ゐることによって 前提から正しい結論を導くことは「計算」と同等になるとも考へた

そのやうな「論理計算」の體系ができたなら そして十分な情報さへあれば 論爭を解決するには「計算してみやう calculemus」と言って答へが出るといふ 夢のやうな構想も書きつけた (Antognazza 2009,244 『形而上學叙説』の少し後の時期)

-----

かういった事情に鑑み 「萬能機械」の構想をライプニッツにまでさかのぼる見解もある (Davis 2011, ch. 1)

したがって 理論的な萬能計算機の可能性を示したアラン・チューリングの計算理論を ライプニッツ解釈に援用しやうといふのは 全然的外れな考へではない

そして 何よりも 萬能計算とプログラム (ないし それが實現する状態遷移函數) の關係を 最低限の數學的道具立てで示せるといふ大きなメリットがある

-----

チューリングが「計算可能性」の本質を解明するための手段として考案した (情報) 機械は 有限個の状態しか持たない制御装置 (つまり 有限オートマトン) と長さに制限のないテープ (つまり 蓄へられる情報に上限がない) とからなる (圖7)

-----

テープ上に「0」の情報が入力されてゐるチューリングマシン
;pic raipuni_fig7
圖7 チューリングマシン

-----

この機械 (制御装置) ができることは (1) テープヘッドがスキャンしてゐるテープの區劃を讀み取る (2) その區劃に「1」を書くか あるいは「0」を書く そして (3) ヘッドを右隣の區劃 あるいは左隣の區劃に移動させるといふ三つしかない

かういふ動作を スキャンした區劃の情報 (入力) に應じて また制御装置を動かすプログラム (すなはち この装置の内部状態を變へていく遷移函數を生み出す) にしたがって自動的に行ふ

-----

問題は テープの區劃上の記号「1」「0」が何を表すのかといふ*コード*を決めなければ 「情報機械」とは言ひがたいといふこと

コードの決め方 (以下「コーディング」と言ふ) はまったく自由なのだが ここでは「テープ上の記号配列で自然數を表す」といふ目的を天下りで導入する

ちなみに 「1・0」による二進法はライプニッツの發見の一つである！

そこで 前述三つの單純な動作だけで この機械に「數を計算する」ことに相當しうるやうな働きをさせることができるか といふのがチューリングの問題提起だった

-----

それで 圖7に示したテープ上に どのやうなコードで自然数を表すかをまづ決めなければならない (異なるコーディングはたくさん可能)

本書で採用するコーディングは 以下の通り

-----

まづ 單一の自然數 n を表すには n+1 個の 1 が連續で並び (その前後はすべて 0 ) テープヘッドは 1 が始まる直前の 0 の區劃にあるものとする

したがって 圖7では 0 が表されてゐる

單一の數だけでなく テープ上には自然數の「順序つき n 個の數」も表現できる

たとへば 0, 1, 2 と三つの數をこの順序で表すには 「1個の 1・0 ・ 2個の 1・ 0・3個の1」その他の區劃は全部 0 で テープヘッドは最初の (左端) 1 の直前の 0 の區劃にある といふコーディングを採用すればよい

-----

そして 重要なことは 「このやうに定めたコードに即してプログラムを書かなければならない」といふことである

コードとプログラムは連動し 不可分である

たったこれだけの事實を押さへるだけで ライプニッツの神にもコーディングが不可欠であることが論証できるのである

-----

チューリングマシンの記述にはいくつか異なる方式があるが 本書では神を「究極のプログラマー」になぞらへたので 「プログラムによる記述」といふ方式を採用したい

これは ハオ・ワンといふ論理學者によって定式化されたもの (Wang 1957)

一般に コンピュータのプログラムはいくつかの基本的な命令が順番に並ぶ系列 (基本命令が種々の組み合はせで n 個連なる) であるが チューリングマシンの場合には 動作が前述のやうに極限まで切り詰められてゐるので 要素となる命令は次の六つしかない

複雑な手續きも すべてこれらの組み合はせで表現される

-----

右: ヘッドをテープの現在の區劃の右へ一コマ移動せよ

左: ヘッドをテープの現在の區劃の左へ一コマ移動せよ

書: 「1」を書け (現在の區劃にすでに「1」があるならそのまま)

消: 「1」を消せ (「0」を書け と同等)

kへ: 現在の區劃に「1」があれば k 番目の命令に移れ ただし なければ順番通り次の命令へ

止: 止まれ

-----

これら基本命令のうちでとくに重要な役割を果たすのは「kへ」といふ条件つき命令で これを上手に使ふことで *同じ手續きを繰り返し適用する*といふ「リカージョン」が可能になる

たとへば 自然數は「プラス1」といふ手續きを繰り返すことで 無限に續くことになるし ある數に同じ數を加へる足し算を何回か繰り返すことで掛け算になる

このリカージョンは 實はプログラムの心臓部分ともなりうるものだが ライプニッツの哲學でも (この言葉こそないものの) きはめて重要な役割を果たしてゐる

いはく 「物質の無限分割」「モナドは生きた鏡」 (モナドが世界を映す 映された世界の中にそのモナドも映る) 「物質のどの階層においても彈性衝突が相互作用のもと」などなど

その威力については 以下の説明で追々明らかになる

-----

讀者 (とくにライプニッツ學者) の誤解を防ぐために注意しておくが わたしは「モナドはチューリングマシンとして解釈できる」と言はうとしてゐるのではない

しかし チューリングマシンの働き方と威力を理解しておかなければ ライプニッツのモナドの働きはたうてい理解できないだらう

なぜなら やがてわかるやうに モナドは無限の世界を映しうるだけでなく それ自體が 有限オートマトンではなく無限オートマトン (内部状態が無限個)になるからである

-----

チューリングマシンによる具體的な計算例はすぐ紹介するが まづ 計算過程を表示する簡便な「記法」をあらかじめ決めておくのが便利だらう

その際 マシン (以下 このやうに略稱) は その動作を決定するプログラム P で代表させることにする

つまり マシンのハードウェアではなく ソフトウェアで機械を代表させるわけである (簡單な限られた動作だけで このマシンはすでに「汎用コンピュータ」の資格を備へてゐる！)

-----

マシンによる「計算」は テープ上に「入力情報」m が與へられた時點からスタートする (圖7參照)

そして 動作が終了した時點で テープ上に殘された記号列 n が「出力情報」である

これを表現するために

m P → n

といふダイヤグラムを使ふ

入力 出力ともに すでに説明したコードによる自然數 あるいは自然數の有限個の組 (數の間はコンマの代はりに 0 で區切られる) である

具體例として 與へられた自然數の「後續數」を計算する過程は次のやうになる

n S → n + 1

このプログラム S は 「どんな數が入力されても後續數を出力する」手續きであることに注意されたい

この条件が満たされて初めて「S は後續數の函數 s(n) = n + 1 を*計算する*」といはれる

これが「マシンによる計算」の*定義*に相當する

そして 「計算可能」とは 「そのやうな計算のプログラムが存在する」といふ意味である

この定義は 日常言語やインフォーマルな算數では意味が多少なりともあやふやな「計算」の本質を 「嚴密な手續きで決まる過程と結果」としての「計算概念」で解明する試みである

もちろん S だけでなく 以下で紹介するプログラムはすべて前述6個の基本命令だけで書き表すことができるが その解説は省略する

モナドロジー

#43

神は現實存在するものの源泉であるばかりか 本質もしくは可能性のなかにある實在的なものの源泉であるのもたしかである

ここでの本質とは [ものがもともともつ]實在としての本質である[1]

なぜなら 神の知性は 永遠眞理もしくは永遠眞理のもとになる觀念が存する領域であり 神がなければ 諸々の可能的なもののなかに實在的なものは何もなくなり 現實存在するものがなくなるばかりか 可能的なものさへなくなってしまふからだ (第二〇節[2])

>[1] 「實在(re'el)」は「現實存在(existant)」と區別される
>ライプニッツはまだ この語をスコラ的な用法で解しており 「實在(re'alite',realitas)」は res(もの) に由來する
>トマス・アクィナスによれば resは 「存在」「一なるもの」などと並ぶ超範疇的概念の一つである
>現實的に存在するものだけでなく 可能的なもの 觀念上のものも res と呼ばれ realitas をもつ (トマス『眞理論』第一問題)

>[2] 『弁神論』第二〇節では 永遠眞理は神の知性の内にはあるが 神の意志からは独立してゐると明言し プラトンが『ティマイオス』(四八a) で 世界の起源が必然性と結びついたヌースにあるとしたことを引いてゐる
>永遠眞理を神の意志とするデカルトとの違ひは根本的である

#44

といふのも 本質すなはち可能性 あるいは永遠眞理のなかに實在性があるならば この實在性はたしかに何か現に存在してゐる現實的なものに基づいてゐるにちがひないのだ

したがって 本質が現實存在を含み 現實的であるためには可能的であれば十分である 必然的な存在の現存に基づいてゐるにちがひない (第一八四[1] ・一八九・三三五節)

>[1] 永遠眞理によって神の存在を示すことはアウグスティヌスに由來する (アウグスティヌス 『自由意志』第二卷第一二・一三・一五章) 
>ライプニッツでは永遠眞理に實在性を與へるのは神の知性であり 神の意志は關はらない(『弁神論』第一八四節)

#45

そこで 神 (すなはち必然的存在) だけが 可能的ならば必ず現に存在する[1]はずだ といふこの特権をもってゐる

そして 限界を含まず否定を含まずしたがって矛盾を含まないものの可能性を妨げるものはないから そのことだけで十分 神の現實存在をア・プリオリ[2]に知ることができる

私たちは 永遠眞理の實在性によってもそれを証明した[3]

-----

しかし私たちはさきほど それをア・ポステリオリにも証明した[4]

なぜなら 偶然的なものは現實存在してゐるが 必然的なもののなかにしかその最後の理由つまり十分な理由をもつことができないのに この必然的なものは自分自身のなかにその現實存在の理由をもってゐるからである

>[1] アンセルムスやデカルトのいはゆる神の存在論的証明は論理的欠陥をもってゐるといふ批判が背景にある
>デカルトはかう論証する
>最も完全な存在者といふ神の觀念は現實存在を含んでゐる
>現實存在は一つの完全性なのだから 神が現實存在を欠いてゐれば最も完全な存在者といふ定義に反するからである
>すなはち 神の本質は存在を含むと
>ライプニッツは この論証では 最も完全な存在者といふ觀念が可能であり矛盾を含んでゐないことが暗黙のうちに仮定されてゐると批判する (『知性新論』第四部第一〇章第七節)
>最大數・最大速度のやうな觀念は不可能な矛盾した觀念だが それとは違って神の觀念は可能的であることを示さなければならない

>[2] 「經驗に先だって」といふ今日の意味ではなく それ以前の意味
>原因から結果を示し 原理から歸結を導くのがア・プリオリな論証で 結果から原因を示し 歸結から原因へ遡るのがア・ポステリオリな論証 (『知性新論』第四部第一七章第一節參照)
>「ア・プリオリに (すなはち經驗によらずに)」といふ用法もある (『形而上學叙説』第八節)

>[3] 第43--44節

>[4] 第36--39節

#46

だが ある人々のやうに 永遠眞理は神に依存してゐるから恣意的であり神の意志による などと思ってはならない

デカルトはさう考へたやうであり その後ポワレ氏[1]がそれに從ってゐる

けれどもこれは偶然的眞理にしか當てはまらない

偶然的眞理の原理は*適合*つまり*最善なもの*の選択である

しかし必然的眞理は 神の知性にのみ依存してその内的對象となってゐるのである[2] (第一八〇--一八四・一八五・三三五・三五一・三八〇節)

>[1] ピエール・ポワレ (1646--1719) はカルヴァン派の神學者
>始めはデカルト派で 『神・魂・および惡の理性的思索』(アムステルダム 1677) を著して ベールから攻撃を受けた
>その後オランダの宗教家アントワネット・ブリニョン (1616-ー1680) の影響で神秘主義者となる

>[2]この點でデカルトとの相違は明らかである
>デカルトは人間の知性と神の知性を異質なものとし 非連續的に捉へる
>ライプニッツでは創造されたものの本質と創造者の本質とのあひだには發展的な差異があるだけで 人間知性の最高原理は神にも人にも共通に當てはまる

#47

そこで 神だけが原初的な一なるもの[1] すなはち本源的な單純實體であり 創造されたすなはち派生的であるモナドは すべてその生産物で いはば神性の不断の閃光放射[2]によって刻々そこから生まれてくる

けれどもこの創造されたモナドは 本質上有限な被造物の受容性のために 制限を受けてゐる (第三八二--三九一・三九八・三九五)

>以前はモナドといふ語は神にも被造物にも適用されてゐたが モナドといふ呼び方を被造物だけに限定したのは 身體をそなへたモナドの特徴 (第一節) を強調するため
